### CI 配置说明
`ci.yml` frontend job 关键步骤：
1. `actions/setup-node@v4`，Node.js 20，`npm` 缓存
2. `npm ci --prefix frontend` — 使用 lock 文件精确安装
3. `npm run lint --prefix frontend` — ESLint 零警告检查
4. `npm test --prefix frontend` — vitest run --coverage，生成 lcov.info
5. `codecov/codecov-action@v4` — 上传 `frontend/coverage/lcov.info`，flag: frontend

## 遇到的问题和解决

1. **问题**：`import React` 在 React 18 + Vite 项目中触发 `no-unused-vars` → **解决**：移除冗余导入，React 17+ 新 JSX Transform 不需要显式导入 React
2. **问题**：`handleCompleteTask` 定义但未使用 → **解决**：在 TaskDetail JSX 中补充"进行中"任务的完成按钮逻辑
3. **问题**：`useEffect` 中调用 `loadTaskDetail` 导致 `react-hooks/exhaustive-deps` 警告 → **解决**：用 `useCallback` 包裹函数并正确声明依赖
4. **问题**：`.eslintrc.cjs` 被 vitest coverage 计入覆盖范围 → **解决**：在 `vite.config.js` coverage exclude 中添加 `**/*.eslintrc.cjs`
