\# HelpMate



校园跑腿/互助平台



\## 项目简介

HelpMate 是一个面向校园用户的跑腿/互助平台，支持发布任务、接单、实时位置、评价系统和钱包支付。



\## 技术栈

\- 前端：React Native / 小程序

\- 后端：Spring Boot 3.x + MyBatis

\- 数据库：MySQL 8.0

\- 部署：Docker Compose
