## 本周工作内容概述

本周我主要负责 HelpMate 校园跑腿平台的前端页面设计。通过 Figma 设计工具完成了三个核心页面的 UI 设计

## 详细工作内容

### 1. 任务详情页（TaskDetail）

#### 设计工作
- 使用 Figma 完成了任务详情页的界面设计
- 设计了任务信息展示区域，包括任务标题、分类、悬赏金额、任务描述等
- 设计了取货地点和送货地点的展示样式
- 设计了发布者信息卡片，包含头像、昵称、评分、已完成任务数等
- 设计了接单按钮和状态标识的交互样式



---

### 2. 个人中心页（UserCenter）

#### 设计工作
- 使用 Figma 完成了个人中心页的界面设计

- 设计了数据统计卡片，展示发布任务数、接受任务数、完成任务数
- 设计了任务列表切换标签（我发布的 / 我接受的）
- 设计了任务列表项的展示样式


---

### 3. 订单消息页（OrderMessage）

#### 设计工作
- 使用 Figma 完成了订单消息页的界面设计
- 设计了订单列表和消息列表的切换标签
- 设计了订单卡片的展示样式，包含订单号、任务标题、状态、进度等
- 设计了订单进度时间轴的展示
- 设计了消息列表项的展示样式
